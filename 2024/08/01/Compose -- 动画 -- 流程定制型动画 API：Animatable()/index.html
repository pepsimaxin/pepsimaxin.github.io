流程定制型动画 API：Animatable()


Animatable 是一个值容器，会在内部储存一个值，它可以在通过 animateTo 更改值时为值添加动画效果。该 API 支持 animate*AsState 的实现。它可确保一致的连续性和互斥性，这意味着值变化始终是连续的，并且会取消任何正在播放的动画。

Animatable 的许多功能（包括 animateTo）以 “挂起函数“ 的形式提供。这意味着，它们需要封装在适当的 “协程作用域“ 内。例如，你可以使用 LaunchedEffect 可组合项针对指定键值的时长创建一个作用域。
